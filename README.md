## Project Path
- App root: `NotionStyleTodoApp/`
- Xcode project: `NotionStyleTodoApp/NotionStyleTodoApp.xcodeproj`

## Current Scope (v1)
- Create task (Return key or + button)
- Edit task inline
- Toggle complete/incomplete
- Swipe to delete
- Assign, update, or clear task due date and time
- SwiftData local persistence
- Empty state: "No tasks"
